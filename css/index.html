  • HTML4基于SGML(更古老), 要写DTD

  • @import是css语法, link是HTML标签,
    @import在页面加载完成加载,
    link没有兼容性问题, @import css2.1+
    link引入的样式权重大鱼@import
    *FOOC 文档样式短暂失效
    原因: 样式加载在html加载之后
    解决: 把样式放在head中
    清楚浮动
    clear:both-> 增加无意义的标签
    overflow: hidden-> 隐藏超出部分
    :after->推荐使用
    父元素设置高度
    overflow:auto->浏览器会自动设置高度,必须定义width,不能定义height, 会产生滚动条
